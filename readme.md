# Flight Simulator
This project is simple 3D flight simulator created with three.js and physi.js libraries.

## Steering
- c - change camera
- w - throttle
- s - brake

#### In air
- Arrow Up - nose down
- Arrow Down - nose up
- Arrow Left - bank left
- Arrow Right - bank right
- , - turn left
- . - turn right

#### On ground
- Arrow Down - nose up
- Arrow Left - turn left
- Arrow Right - turn right
